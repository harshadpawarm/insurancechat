
# Project Blueprint: Insurance Chatbot

## Overview

This document outlines the plan, design, and features of the Insurance Chatbot, a web application built with Angular and Firebase. The chatbot will provide users with information about insurance policies, answer frequently asked questions, and offer a seamless conversational experience.

## Style, Design, and Features

### Initial Version (v1)

*   **Framework:** Angular v20+
*   **Architecture:** Standalone components
*   **State Management:** Signals
*   **Styling:** Modern CSS with a clean and intuitive interface
*   **Backend:** Firebase (Authentication, Firestore, Cloud Functions)
*   **Core Features:**
    *   User authentication (anonymous)
    *   Real-time chat interface
    *   Conversation history stored in Firestore
    *   Simple Q&A knowledge base in Firestore
    *   Cloud Function for basic chatbot logic (keyword matching)

## Current Plan: Phase 1

### 1. Project Setup and UI Skeleton

*   **Create `blueprint.md`:** Document the project's overview, design, and features.
*   **Firebase Project Setup:** Create a new Firebase project and configure Authentication, Firestore, and Cloud Functions.
*   **Angular Project Setup:** Set up the Angular project with the necessary dependencies.
*   **UI Skeleton:** Create the basic UI for the chatbot, including the chat window, message input, and send button.
